# WDIO Electron Framework Documentation Pack

This folder contains Confluence-ready Markdown documentation for the WDIO Electron Framework.

Recommended Confluence page structure:

1. `Framework Documentation`
   - Source: `framework-documentation.md`
2. `Architecture and Flow Diagrams`
   - Source: `architecture-diagrams.md`
3. `Test Authoring Guide`
   - Source: `test-authoring-guide.md`
4. `Allure Evidence and Step Reporting`
   - Source: `allure-evidence-and-steps.md`

## Upload Notes

- Paste each Markdown file into a separate Confluence page.
- If the Confluence instance supports Mermaid, paste the Mermaid blocks directly into the Mermaid or diagram macro.
- If Mermaid is not supported, export each diagram from a Mermaid renderer as PNG/SVG and attach it to the Confluence page.
- Keep this folder versioned with code changes so framework behavior and documentation stay aligned.
